Follow these steps to install and run K-HOLE on your machine.

1. Clone the Repository

git clone https://github.com/KetaGod/K-Hole.git
cd khole

2. Install Python Requirements

pip install -r requirements.txt

3. Launch the Toolkit

python khole.py

4. Optional Tools

  • Install Google Chrome + ChromeDriver (for headless browser emulation)
  • TOR service if using --torcycle
